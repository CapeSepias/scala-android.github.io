About

Easy-to-use plugin for existing and newly created android projects. It is tested and developed against sbt >= 0.13.6

3rd party libraries can be included by placing them in libs as in regular projects, or they can be added by using sbt's libraryDependencies feature.

Plugin supports normal Android projects and projects that reference library projects.

100% compatible with Google's own Gradle build system due to the use of the same underlying com.android.tools.build:builder implementation.
